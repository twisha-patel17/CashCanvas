# 💰 CashCanvas

### Personal Finance, Kept Honestly.

A modern Personal Finance Management Application built using the MERN Stack that helps users track income, monitor expenses, manage monthly budgets, and visualize spending habits through beautiful financial analytics.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/twisha-patel17/CashCanvas.git
```

### Move into the Project

```bash
cd CashCanvas
```

---

### Frontend Setup

```bash
cd client

npm install

npm run dev
```

---

### Backend Setup

```bash
cd server

npm install

npm run dev
```
